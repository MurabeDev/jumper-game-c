# Jumper Game
This is simple and endless runner in C using [raylib](https://raylib.com)
# Feautures
- Score and High Score
- Simple gameplay
- Game has a simple architecture, making it easy to add something new
# How to play 
- ```SPACE``` - jump and retry
- ```ESC``` - close game
# How to build
#### Windows
 Via Visual Studio (or other IDE)
- Download repository
- Open .sln file
- Build it (F5 -> Build and Run with debugger)
